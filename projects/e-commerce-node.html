E-Commerce System

  • Tech Stack: Node.js, Firebase, Javascript, Express, Heroku
  • Github URL: Project Link

I developed an e-commerce system using Node.js and Express. The system consists of a store manager application, a user application, an application for the delivery person, and a control panel website.

The data was stored on the firestore in the firebase.

The functions I did are the following:

  • Functions for logging in, creating an account, modifying the profile, and verifying the email.
  • CRUD methods for stores, sections, stores, items, and products .
  • A follower that repeats itself on the server in order to process operations for generating a discount code .
  • CRUD methods on offers for products, in addition to a method that repeats itself on the server to delete the offer if it exceeds the specified period.
